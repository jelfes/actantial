# actantial

`actantial` is a research tool for analysing narratives using Greimas' Actantial Model. It uses LLMs to annotate texts with six character roles — called actants — including Subject, Object, Sender, Receiver, Helper, and Opponent.

See the [Background](background.md) page for an introduction to the Actantial Model, or head straight to the [Reference](reference/runner.md) for the full API documentation.

For installation and a quick start, see the [README](https://github.com/jelfes/actantial).
